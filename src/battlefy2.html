Battlefy

Streamlining the Tournament Creation Flow

Problem

Tournament creation is an integral part of the Battlefy platform. There are multiple steps to the wizard and many tournament organizers find the tournament creation process complicted and confusing.

  1. Many advanced features are only utilized by power users.
  2. It is not clear to users that you can create a tournament for any game as they are not aware of the search function for games.

Solution

To streamline the tournament creation process, we clarified which fields were required and which were advanced.

Users could not see the search function in the game selection section of the tournament creation wizard. After trying different ways to place the search feature, we finally made it visible by placing it in the center of the page. This way, users knew that they can simply search for any game and quickly move onto the next section.

Impact

Tournament creation is a core part of the platform. Organizers create tournament so that players can play in them. It is critical to make this process as simple as possible for organizers and help them in creating tournaments.

As a result of this project:

  • When users reached this page, they were searching more. We saw a 33% increase in searches for games.
  • Users were finding the games they came for. There was a 13% increase in selecting a game after searching.
